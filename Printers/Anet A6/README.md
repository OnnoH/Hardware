## Anet A6 profile for Cura 3.5

- Start Cura once
- Create a custom printer called "Anet A6" without changing any of the settings
- Close Cura
- Save and overwrite both files in "AppData\Roaming\cura\3.1\definition_changes"
- Start Cura again.
